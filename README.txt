تشغيل الموقع:
1) افتح ملف index.html داخل أي متصفح حديث.
2) جميع صور المعرض وصور الشهادات المرفقة تم وضعها داخل المجلد assets وربطها مباشرة بالموقع.
3) الشهادات الأكاديمية الأساسية بقيت كنصوص حالياً ويمكن إضافة صورها لاحقاً داخل script.js أو بنفس البنية الحالية.
4) لتبديل أي صورة لاحقاً، استبدل الملف داخل assets مع الحفاظ على نفس الاسم، أو عدّل المسار داخل script.js.
5) تم استخدام Gallery_4.jpeg مؤقتاً كصورة رئيسية في واجهة الموقع لعدم توفر صورة بورتريه منفصلة ضمن الملفات الحالية.

Run instructions:
1) Open index.html in any modern browser.
2) All provided gallery and certificate images are now stored in the assets folder and linked directly to the site.
3) Academic certificates remain text-based for now and can be updated later in script.js.
4) To replace any image later, keep the same filename inside assets or update the path in script.js.
5) Gallery_4.jpeg is currently used as the hero image because no separate portrait image was provided.
