تشغيل الموقع:
1) افتح ملف index.html داخل أي متصفح حديث.
2) الصور الحالية موجودة داخل مجلد assets.
3) يمكن إضافة صور المعرض لاحقاً داخل المجلد نفسه ثم تعديل script.js.
4) يمكن إضافة صور الشهادات أو تفاصيلها لاحقاً داخل قسم Certificates في script.js.

Run instructions:
1) Open index.html in any modern browser.
2) Current images are inside the assets folder.
3) Gallery images can be added later in the same folder, then update script.js.
4) Certificate images/details can be added later in the Certificates section inside script.js.
